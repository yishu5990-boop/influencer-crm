# 达人邮件管理助手

一站式管理 TikTok 达人联系、邮件沟通与合作阶段的全栈 Web 应用，内置 AI 智能分析与自动发现达人能力。

## 核心功能

- **达人管理** — 添加、搜索达人，记录报价、品牌、合作阶段，支持手动录入和邮箱批量发现
- **邮件同步** — 通过 Gmail IMAP 同步真实收件箱，自动匹配达人并归入对应时间线
- **AI 智能分析** — 自动生成邮件中文摘要，根据往来内容推荐合作阶段变更
- **自动发现达人** — 扫描收件箱，通过关键词匹配识别潜在达人，用户勾选后一键导入
- **多邮箱管理** — 支持绑定多个 Gmail 邮箱，每个邮箱独立同步，应用专用密码或 OAuth 均支持

## AI 技术说明

- **模型**：DeepSeek Chat API（`deepseek-chat`）
- **邮件摘要**：Prompt 要求模型从邮件中提取核心信息，用 1-2 句中文概括沟通要点
- **阶段建议**：传入达人当前阶段 + 上下文邮件内容，模型分析沟通进度并推荐下一阶段（如「初洽 → 提报待审核」），同时给出理由
- **达人发现**：不依赖 AI，使用两轮关键词打分（先主题、后正文）筛选潜在达人，零成本且速度快

## 本地运行

```bash
# 1. 进入后端目录，安装依赖
cd server
npm install

# 2. 配置 server/.env
cp .env.example .env
# 必填：JWT_SECRET, GMAIL_EMAIL, GMAIL_APP_PASSWORD
# 可选：DEEPSEEK_API_KEY, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET

# 3. 启动后端 (端口 3001)
node src/index.js

# 4. 进入前端目录，安装依赖
cd ../influencer-crm
npm install

# 5. 配置前端 .env
echo "VITE_API_BASE_URL=http://localhost:3001/api" > .env

# 6. 启动前端开发服务器
npm run dev
```

访问 `http://localhost:5173`，点击「一键演示登录」即可体验。

> 线上演示地址：https://hys-crm.top（通过 Cloudflare Tunnel 部署）
